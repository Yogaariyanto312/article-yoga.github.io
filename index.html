MATERI HTML

A. Pengertian HTML

      HTML atau Hyper Text Markup Language merupakan sebuah bahasa permrograman terstruktur yang dikembangkan untuk membuat halaman website yang dapat diakses atau ditampilkan menggunakan Web Browser. HTML sendiri secara resmi lahir pada tahun 1989 oleh Tim Berners Lee dan dikembangkan oleh Word Wide Web Consortium (W3C), yang kemudian pada tahun 2004 dibentuklah Web Hypertext Application Tecnology Working Group (WHATG) yang hingga kini bertanggung jawab akan perkembangan bahasa HTML ini. Hingga kini telah mengembangkan HTML 5, sebuah versi terbaru dari HTML yang mendukung tidak hanya gambar dan teks, namun juga menu intraktif,audio,video, dan lain lain sebagainya.
